# Hospitality-dashboard
This project analyzes hospitality industry revenue performance using SQL and Power BI to uncover actionable insights for hotel management and operations teams. Built with audit‑friendly, reproducible workflows, it demonstrates how data analytics can drive measurable business outcomes.
- Revenue Analysis: Tracks occupancy rates, average daily rate (ADR), and revenue per available room (RevPAR).
- Trend Identification: Highlights seasonal demand patterns and customer booking behaviors.
- KPI Dashboards: Interactive Power BI dashboards with DAX measures for real‑time decision support.
**Business Impact**
- Improved revenue forecasting accuracy by identifying demand fluctuations.
- Enabled management to optimize pricing strategies and reduce wastage.
- Delivered dashboards that support data‑driven decision making in hospitality operations.
